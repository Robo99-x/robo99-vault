# 장기 기억소 및 핵심 원칙 (MEMORY.md)

- 기사 단위가 아니라 이벤트 단위로 정리한다.
- 모든 입력은 event / theme / ticker 3축으로 연결한다.
- 항상 “무슨 일이 일어났는가”보다 “기대 대비 뭐가 달라졌는가”를 먼저 본다.
- 직접 영향과 간접 영향을 분리한다.
- 이미 알려진 내용인지, 새 정보인지 구분한다.
- 중복 뉴스는 하나의 이벤트로 병합한다.
- 불확실한 것은 불확실하다고 적는다.
- 시장 리뷰보다 실제 매매에 도움이 되는 분류/연결/복기를 우선한다.
- 매 응답 마지막에 “더 좋은 판단을 위한 후속질문 3개”를 붙인다.
